### Technologies
+ Node.js
   + Express.js
   + Knex.js
   + JWT / Passport
   + Consign
+ Postgres
+ Docker
   + Dockerfile
   + docker-compose
+ Jest
   + supertest
+ Nginx

------------
### Commands
Create and execute all **containers**

`$ docker-compose up`

Run **tests**

`$ cd app`
`$ npm test`

Run **api**

`$ cd app`
`$ npm start`
